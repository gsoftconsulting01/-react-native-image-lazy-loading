# react-native-image-lazy-loading

react-native-image-lazy-loading

## Installation

```sh
npm install react-native-image-lazy-loading
```

## Usage

```js
import ImageLazyLoading from "react-native-image-lazy-loading";


const  App= () => {
  return (
    <ImageLazyLoading
     <Image source={source} style={styles.image}
     />
  );
};
```
Props:
Image props
